# Getting Started — Onboarding Guide

## Step 1: Create Your Workspace
After signing up, create a workspace for your team:
1. Click 'New Workspace' from the dashboard
2. Enter workspace name and invite team members by email
3. Set permissions: Admin, Editor, or Viewer

## Step 2: Connect Your Data Sources
Supported integrations:
- REST APIs (using API key authentication)
- CSV and Excel file uploads
- Google Sheets (OAuth connection)
- PostgreSQL and MySQL databases

## Step 3: Create Your First Project
1. Click 'New Project' in your workspace
2. Select a data source
3. Choose a template or start from scratch
4. Save and share with your team

## Step 4: Set Up Notifications
Go to Settings > Notifications to configure:
- Email alerts for system events
- Slack integration for real-time alerts
- Webhook callbacks for custom systems
