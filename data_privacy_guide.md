# Data Export and Privacy Guide

## Exporting Your Data
You can export all your data at any time:
1. Go to Settings > Data Management > Export
2. Select data types: Projects, Reports, User activity, Audit logs
3. Choose format: CSV, JSON, or PDF
4. Click 'Request Export' — large exports are emailed within 2 hours

## Data Retention Policy
- Active accounts: Data retained indefinitely
- Cancelled accounts: Data retained 90 days then permanently deleted
- Deleted accounts: Data removed within 30 days
- Audit logs: Retained for 1 year regardless of account status

## Data Privacy and GDPR
We are GDPR compliant. As a data subject you have the right to:
- Access your personal data (submit request via Settings > Privacy)
- Correct inaccurate data
- Delete your data (right to erasure)
- Data portability — receive your data in machine-readable format
- Object to processing

For GDPR requests: privacy@example.com. Response within 30 days.

## Data Security
- All data encrypted at rest using AES-256
- Data in transit encrypted using TLS 1.3
- Regular third-party security audits conducted
- SOC 2 Type II certified
