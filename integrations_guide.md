# Integrations and Third-Party Connections

## Available Integrations

### Slack
Connect Slack to receive alerts and run commands:
1. Go to Settings > Integrations > Slack
2. Click 'Connect to Slack' and authorize
3. Choose a channel for notifications
4. Use /support command in Slack to create tickets

### Google Workspace
Sync with Google Drive and Sheets:
1. Settings > Integrations > Google
2. Sign in with Google and grant permissions
3. Browse and import Google Sheets directly

### Zapier
Connect with 5000+ apps via Zapier:
- Use our Zapier trigger: 'New Event in Example App'
- API key authentication required
- Find us by searching 'Example App' in Zapier

### Webhooks
Send real-time event data to your systems:
1. Settings > Integrations > Webhooks > Add Webhook
2. Enter your endpoint URL (must be HTTPS)
3. Select events to subscribe to
4. Save — we send a test POST request to verify

## Troubleshooting Integrations
- Disconnected integration: Re-authorize from Settings > Integrations
- Missing data: Check the sync log for errors
- Rate limits: Webhook deliveries are capped at 500/minute
